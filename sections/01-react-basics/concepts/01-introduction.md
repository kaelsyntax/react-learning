# Introduction to React

## What is React?

React is a JavaScript library used to build user interfaces (UI), especially for web applications.

With React, you create interfaces by combining reusable pieces called components.

React can run on the client (browser) and also on the server with tools like server-side rendering.

## Declarative Approach

React is declarative: you describe how the UI should look for a given state, and React updates the DOM for you.

This makes code easier to read, maintain, and debug.

## Component-Based Architecture

A React app is built from components.

Each component can:

- receive data through `props`
- manage internal data through `state`
- be reused across different parts of the app

This helps you build scalable and organized applications.
