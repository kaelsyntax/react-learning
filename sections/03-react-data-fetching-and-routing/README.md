# 03 React Data Fetching and Routing ⚡

Build practical React skills for data-driven UIs and client-side navigation.

---

## 🧠 Overview

This section focuses on building real-world React patterns for handling asynchronous data and navigation.

Instead of isolated exercises, concepts are applied to a single evolving project to simulate real application development.

---

## 🎯 What You’ll Learn

- Managing async data with `useEffect`
- Handling UI states (loading, error, empty, success)
- Building controlled search experiences
- Avoiding race conditions and unnecessary requests
- Structuring reusable logic with custom hooks
- Understanding client-side routing fundamentals
- Working with dynamic routes and code-splitting
